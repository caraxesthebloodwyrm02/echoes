## Escalation & Support

### If Board Member Unresponsive
1. **Week 1:** Send reminder email
2. **Week 2:** Follow up with phone call
3. **Week 3:** Discuss with project lead
4. **Week 4:** Identify backup expert if needed

### If Conflicting Advice
1. Document different perspectives
2. Convene synchronous discussion (if feasible)
3. Seek synthesis or clarification
4. Make informed decision with rationale
5. Communicate decision back to board

### If Additional Expertise Needed
1. Identify specific gap
2. Ask board for referrals
3. Reach out to candidate expert
4. Brief with context and constraints
5. Integrate insights into project

**Purpose:** Facilitate understanding across fields

**Examples:**
| Concept (Origin) | Analogy (Target Field) | Insight |
|------------------|----------------------|---------|

**Updated:** Each advisory board meeting

### Integrative Workshops
**Frequency:** Once per phase (optional)
**Duration:** Half-day

**Format:**
1. **Cross-Pollination:** Each expert presents core concepts from their field
2. **Problem Framing:** Reframe project challenges through each lens
3. **Synthesis:** Identify common patterns and insights
4. **Application:** How insights inform project design
